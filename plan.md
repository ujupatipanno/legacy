# 레거시 파일 플러그인
## 규칙
- 모든 UI는 obsidian 기본 API로만 구현한다.
- 모든 UX는 한국어 사용자 기준으로 구현한다.
- 절대 인라인 스타일을 사용하지 않고 obsidian native class만 사용한다.
### 구현 방안
- 명령어 추가: 현재 문서의 레거시 파일 만들기
	- 현재 노트의 사본을 생성하고 사본의 이름을 `{{note.name}}_legacy_YYYYMMDDHHmm`으로 만들기.
	- 설정 탭에서 단순 텍스트 필드로 사본이 저장되는 폴더 위치를 지정하기.
		- 기본값은 legacy 폴더
    - 아이콘은 `lucide-save`를 사용하기
- 명령어 추가: 현재 문서의 레거시 파일 보존하기
	- 현재 노트의 레거시 파일들을 오래된 문서부터 최신 문서 순으로 나열한 하나의 문서를 생성하기. 이때 노트가 바뀔 때마다 `___` 구분선을 넣고 노트 맨 위에 `#{{note.name}}_legacy_YYYYMMDDHHmm`라는 태그를 달아서 서로 구분한다.
		- 이렇게 생성된 노트의 제목은 `{{note.name}}_legacy_(시작일)YYYYMMDDHHmm-(종료일)YYYYMMDDHHmm`로 한다.
	- 이 보존 노트가 생성이 잘 완료되면 여러 레거시 파일들을 모두 삭제한다.
    - 아이콘은 `lucide-save-all`를 사용하기
	- 설정 탭에서 오름차순 내림차순을 변경할 수 있게 한다.
		- 옵션은 드롭다운으로 처리한다.
- 리본 메뉴에도 두 명령어를 추가한다.
